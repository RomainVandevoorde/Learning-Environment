Tim Berners-Lee

Inventor of HTML

Sir Timothy John Berners-Lee, is an English engineer and computer scientist, best known as the inventor of the World Wide Web. He made a proposal for an information management system in March 1989, and he implemented the first successful communication between a Hypertext Transfer Protocol (HTTP) client and server via the internet in mid-November the same year.
